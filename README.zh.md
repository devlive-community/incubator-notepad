### 什么是 NoteRunner?

---

NoteRunner 是一个开源软件，使用 Rust 语言进行开发。它是一款简洁的文本编辑器，适用于各种操作系统平台。该软件的设计理念是提供一个轻量级、高效的编辑环境，以满足用户对于编辑文本文件的基本需求。
